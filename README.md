# Web3 Starter

### Stack

- Next.js
- Chakra UI
- Storybook
- Wagmi (includes React Query)
- React Hook Form (+ zod schema resolver)

### Getting started

```bash
# Run Next.js dev server
yarn dev

# Run Storybook
yarn storybook
```
